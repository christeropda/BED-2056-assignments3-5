Go to: https://www.cdc.gov/nchs/data_access/vitalstatsonline.htm

Dowload the datasets for 2018 and 2019
------------------------------------------------------------------------------------------------------

IMPORTANT!!!
Extract the files in the same directory as the python file, and rename the 2018 dataset to "2018.txt",
similarly you should rename the 2019 dataset as "2019.txt".

------------------------------------------------------------------------------------------------------

Run by entering in terminal:
python3 ass7.py

make sure matplotlib is installed!

-------------------------------------------------------------------------------------------------------

If errors occour, its probably because the path is wrong.

the treestructure should look like this:

folder                                                     (folder containing all documents)
------ ass7.py                                             (executable)
------ Nat2018us                                           (folder)
---------------- 2018.txt                                  (txt file)
------ Nat2019us                                           (folder)
---------------- 2019.txt                                  (txt file)

------------------------------------------------------------------------------------------------------
